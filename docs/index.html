Introduction

LogYourBody is a modern body composition tracking application built with React, TypeScript, and Supabase. This documentation provides an overview of setup, usage, and key features.

Installation

  1. Clone the repository.
  2. Install dependencies with npm install.
  3. Copy .env.example to .env and fill in your environment variables.
  4. Run npm run dev to start the development server.

Usage

Once the development server is running, open http://localhost:5173 to view the app. Use your Supabase credentials to sign in and start logging your body metrics.

Features

  • Body composition tracking with interactive charts.
  • Subscription-based premium analytics and unlimited storage.
  • Responsive design with offline support.
  • Secure authentication powered by Supabase.
